More Red is a mod for the Minecraft game which uses the Neoforge modloading platform.

[You can download More Red on Curseforge!](https://www.curseforge.com/minecraft/mc-mods/more-red)

More Red adds these things:

Wires and cables!

![](https://commoble.s3.us-east-1.amazonaws.com/images/morered/blockless_bundled_cables.png)

Redstone logic gates!

![](https://commoble.s3.us-east-1.amazonaws.com/images/morered/xnor_gate_combinations.png)

Move things from where they shouldn't be to where they should!

![](https://commoble.s3.us-east-1.amazonaws.com/images/tubesreloaded/tubing_pliers.png)

All content is described in detail in the sidebar.
