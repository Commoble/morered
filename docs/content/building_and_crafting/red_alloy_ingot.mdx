## Usage

Red Alloy Ingots are used to craft:
* [Red Alloy Wire](/en/mod/morered/docs/content/wires_and_cables/red_alloy_wire)
* [Redwire Junctions](/en/mod/morered/docs/content/wires_and_cables/redwire_junction)
* [Redwire Posts](/en/mod/morered/docs/content/wires_and_cables/redwire_post)
* [Redwire Relays](/en/mod/morered/docs/content/wires_and_cables/redwire_relay)
