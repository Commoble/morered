## Usage

While the filter holds no item, it acts as a [Shunt](/en/mod/morered/docs/content/transportation/shunt).

Right-clicking the filter will open a menu with a single item slot.
While that slot holds an item, it will be displayed inside the filter,
and only items of that type can be inserted into the filter.

![](https://commoble.net/images/tubesreloaded/filter_2.png)

Using the [Pliers](/en/mod/morered/docs/content/transportation/pliers) or another mod's wrenchlike item on a filter will rotate it.
